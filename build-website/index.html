Desenvolvedor Full-Stack com uma abordagem diferenciada de Engenharia de Software. Minha especialidade é construir soluções de ponta a ponta, escaláveis e com código de alta qualidade, unindo um Back-End robusto em Java (Spring Boot), Node.js e SQL/NoSQL, com interfaces Front-End modernas desenvolvidas com Dart (Flutter) e JavaScript.

Possuo profundo domínio e aplicação prática de Arquitetura Limpa (Clean Architecture), SOLID e TDD, o que garante sistemas altamente testáveis e de fácil manutenção. Minha base em Engenharia me conferiu uma visão analítica e sistêmica crucial para resolver desafios complexos. Essas habilidades foram aplicadas em projetos de grande escala, como a criação de funcionalidades para o sistema SIGAA, e no desenvolvimento de aplicativos mobile complexos.
